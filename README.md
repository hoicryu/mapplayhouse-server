## 소개

- 극단 동호회원들을 위한 커뮤니티 웹앱입니다.
- 모집중인 작품 기수에 지원할 수 있습니다.
- 연습실을 시간대별로 대관 신청 할 수 있습니다.
- 본인이 속한 기수의 사진, 영상 및 자료조회가 가능합니다.

## 개발환경

- FrontEnd: HTML, React, Framwork7, Tailwinds, Recoil
- BackEnd: RubyOnRails, PostgreSQL, Activeadmin, Redis
- 서비스 배포 환경: Vercel(예정)
